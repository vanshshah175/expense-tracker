PennyWise - Multi-Account Expense Tracker
A simple, responsive, and fully functional single-page web application to track expenses and income across multiple accounts. Built with vanilla HTML, CSS, and JavaScript.

Features
Multi-Account Support: Manage different accounts (e.g., Wallet, Bank Account, Savings) independently.
Transaction Management: Add and delete income or expense transactions easily.
Dashboard Overview: View real-time summaries for Total Balance, Income, and Expenses for the selected account.
Data Persistence: All data is saved automatically to your browser's LocalStorage, so your records remain even after refreshing the page.
Custom Categories: Tailored specifically for tracking the following categories:
Income: Salary, Investment
Expense: Food, Drink, Travel, Utilities, Milk
Responsive Design: Works seamlessly on desktop and mobile devices.
Getting Started
No installation or complex setup is required. This is a standalone HTML file.

Download: Save the index.html file to your computer.
Run: Double-click the file or drag it into your web browser (Chrome, Firefox, Safari, Edge).
Use: Start creating accounts and tracking your expenses!
How to Use
1. Create an Account
On the left sidebar, click "+ Add New Account".
Enter a name (e.g., "My Wallet"), an initial balance, and choose an icon.
Click Create Account.
2. Add a Transaction
Select the account you want to use from the left sidebar.
In the "Add Transaction" card on the right, toggle between Expense and Income.
Fill in the Description and Amount.
Select a Category from the dropdown list.
Click Add.
3. View History & Manage
View recent transactions in the Recent History list.
Click the (X) icon next to any transaction to delete it (the account balance will be updated automatically).
Switch accounts using the sidebar to see balances for different wallets or banks.
